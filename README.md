# Flappy Lane Racer

A tiny browser game that mixes Flappy Bird timing with a three-lane car simulator.

## Play

Open `index.html` in a browser, or host this folder with GitHub Pages.

## Controls

- Space or Up Arrow: boost upward
- Left Arrow and Right Arrow: change lanes
- On mobile: use the on-screen buttons

## Modes

- Easy: slower traffic and wider gaps
- Medium: faster traffic
- Hard: tighter gaps and heavier speed
